# Loragica Trademark & Brand Assets Policy

**Effective Date:** June 19, 2026

---

## Why Is This Document Separate from the Code License?

The source code in this repository is **open source** (AGPL-3.0) — free to use, modify, and redistribute in accordance with its license terms. However, **the open-source license for the code does not automatically grant the right to use Loragica's name and brand identity.**

This is a standard practice used by many large open-source projects (such as Mozilla Firefox, WordPress, Audacity) — the code can be freely forked, but the official name and logo remain protected to ensure users are not confused about what is an official Loragica product and what is not.

## 1. What Is Protected

The following Loragica brand assets are **fully protected** and **may not be used without prior written permission** from Loragica:

- The name **"Loragica"** and all forms of its spelling/variations
- The asymmetric 2×2 grid logomark and all its variations (horizontal, stacked, superscript)
- The Loragica wordmark in any typography
- The brand color palette: **Logos Blue** `#185FA5`, **Horasis Teal** `#1D9E75`, **Agora Purple** `#534AB7`
- The tagline **"Learn · See · Connect"**
- The three-pillar philosophy of **Logos (λόγος) · Horasis (ὅρασις) · Agora (ἀγορά)** as a brand identity system
- The brand typography combination (Space Grotesk + Plus Jakarta Sans + JetBrains Mono) as part of Loragica's visual identity

## 2. What You Cannot Do

Without explicit written permission from Loragica, you **cannot**:

- ❌ Use the name "Loragica" for any other project, product, or service — including forks of this code
- ❌ Use the Loragica logo, logomark, or wordmark outside of official Loragica materials
- ❌ Create a derivative project (fork) from this code and publish it under the same or similar name or visual identity as Loragica
- ❌ Use the combination of Loragica's three brand colors along with the Logos–Horasis–Agora pillar system as a brand identity for other projects
- ❌ State or imply any affiliation, endorsement, or official relationship with Loragica where none exists

**If you fork this code to build a similar platform, you are required to change the name, logo, and all visual identity elements to your own.**
