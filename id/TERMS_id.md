## 1. Tentang Dokumen Ini

Dengan mengakses atau menggunakan situs web Loragica ([loragica.com](https://loragica.com) / [loragica.id](https://loragica.id)) ("Situs") atau kode sumber di repository ini, kamu setuju untuk terikat dengan Syarat dan Ketentuan ini. Jika kamu tidak setuju, mohon untuk tidak menggunakan Situs atau kode sumber ini.

Loragica saat ini berada pada **tahap branding dan validasi awal**: belum ada sistem akun pengguna, belum ada transaksi pembayaran, dan belum ada platform pembelajaran penuh yang aktif. Syarat ini akan diperluas seiring berkembangnya produk.

---

## 2. Penggunaan Situs

Situs ini disediakan untuk tujuan informasi mengenai Loragica sebagai inisiatif EdTech STEM. Kamu setuju untuk **tidak**:
- Menggunakan Situs untuk tujuan ilegal atau melanggar hukum yang berlaku di Indonesia
- Mencoba mengakses bagian non-publik dari sistem kami tanpa izin
- Menyalahgunakan formulir kontak/pendaftaran untuk spam atau penipuan
- Mengklaim afiliasi resmi dengan Loragica tanpa izin tertulis

## 3. Lisensi Kode Sumber (Open Source)

Kode sumber yang terdapat dalam repository ini dilisensikan di bawah **GNU Affero General Public License v3.0 (AGPL-3.0)**. Teks lengkap lisensi tersedia di file [`LICENSE`](./LICENSE).

Secara singkat, AGPL-3.0 berarti:

| Kamu boleh | Dengan syarat |

| ✅ Menggunakan kode untuk keperluan apa pun, termasuk komersial | 📋 Source code turunan tetap harus dilisensikan AGPL-3.0 |
| ✅ Mempelajari dan memodifikasi kode | 📋 Perubahan signifikan harus didokumentasikan |
| ✅ Mendistribusikan ulang kode (asli atau modifikasi) | 📋 Sertakan salinan lisensi & notice copyright |
| ✅ Menjalankan versi modifikasi sebagai layanan jaringan/web | 📋 **Wajib** menyediakan source code lengkap ke pengguna layanan tersebut |

Poin terakhir adalah ciri khas AGPL dibanding lisensi open source lain (seperti MIT atau GPL biasa): jika kamu menjalankan versi modifikasi dari kode ini di server publik (misalnya untuk membuat platform serupa), kamu **wajib** memberi akses source code-nya kepada pengguna platform tersebut — bukan hanya kepada orang yang menerima salinan filenya secara langsung.

Lisensi ini berlaku untuk **kode sumber**, bukan untuk brand assets Loragica. Lihat bagian 4.

## 4. Brand Assets — Tidak Termasuk Lisensi Open Source

Lisensi AGPL-3.0 di atas **hanya berlaku untuk kode**. Hal-hal berikut **tidak** termasuk dalam lisensi open source dan tetap menjadi hak eksklusif Loragica:

- Nama "**Loragica**" dan segala variasinya
- Logomark/logo grid 2×2 dan seluruh aset visual identitas
- Wordmark dan tipografi brand
- Sistem warna brand (Logos Blue, Horasis Teal, Agora Purple)
- Tagline "**Learn · See · Connect**"
- Konsep tiga pilar Logos–Horasis–Agora sebagai identitas brand

Detail lengkap mengenai penggunaan brand assets ada di [`TRADEMARK.md`](./TRADEMARK.md). **Singkatnya: kamu boleh fork dan memodifikasi kodenya, tapi tidak boleh memakai nama atau identitas visual Loragica untuk project hasil fork tersebut.**
